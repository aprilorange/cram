## Development

**prepare:** node.js/gulp installed, then `npm i`

- `qna/cson` 题库目录
- `src/styl` 样式文件目录
- `src/js` JS 文件目录
- `src/html` HTML 文件目录

修改或添加后需要 `gulp build`
本地演示执行 `gulp`
